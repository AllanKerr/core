# Focus Trap

## d2l-focus-trap

```html
<script type="module">
  import '@brightspace-ui/core/components/focus-trap/focus-trap.js';
</script>

<d2l-focus-trap>
  content in which focus should be trapped
</d2l-focus-trap>
```

**Properties:**

- `trap` (Boolean): Whether the focus trap is actively trapping focus
